# Star Citizen Cockpit Pro FR

> **Le cockpit communautaire francophone pour Stream Deck dédié à Star Citizen**

---

## 📖 Présentation

**SCCP-FR** (Star Citizen Cockpit Pro FR) est un projet open source visant à créer le cockpit Stream Deck francophone de référence pour Star Citizen.

L'objectif est de proposer une interface moderne, ergonomique et entièrement en français, permettant aux pilotes de contrôler leur vaisseau rapidement sans avoir à mémoriser des dizaines de raccourcis clavier.

Le projet est conçu pour être :

* 🇫🇷 Entièrement en français
* 🎮 Compatible avec Star Citizen
* 🎛️ Optimisé pour Stream Deck XL
* 🆓 Gratuit
* 🔓 Open Source
* 📚 Documenté
* 🔄 Facile à faire évoluer

---

# 🎯 Objectifs

Créer une solution complète comprenant :

* Un profil Stream Deck XL
* Des profils simplifiés (Débutant, Standard et Expert)
* Une bibliothèque d'icônes HD
* Une documentation complète
* Des macros
* Une base de données des commandes Star Citizen
* Un guide d'installation
* Un guide utilisateur
