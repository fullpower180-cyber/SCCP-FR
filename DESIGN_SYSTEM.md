# 🎨 SCCP-FR DESIGN SYSTEM

**Projet :** Star Citizen Cockpit Pro FR

**Version :** 0.1.0 Foundation

**Statut :** En développement

---

# 🎯 Philosophie

Le Design System SCCP-FR définit les règles graphiques et ergonomiques du projet.

Tous les éléments créés (icônes, logos, bannières, profils Stream Deck et documentation) devront respecter ces règles afin de garantir une expérience cohérente.

Les principes du projet sont :

* Simplicité
* Lisibilité
* Ergonomie
* Immersion
* Cohérence
* Évolutivité

# 🖼️ Format des icônes

## Source

* SVG

## Export

* PNG 128 × 128
* PNG 256 × 256

## Fond

Transparent

---

# 📐 Structure d'une icône

Toutes les icônes devront utiliser la même structure :

* Cadre identique
* Pictogramme centré
* Couleur de catégorie
* Bandeau inférieur
* Nom en français

---

# 🔤 Convention de nommage

Chaque icône possède un identifiant unique.

Exemple :

SCFR-001

Nom :

Prêt au vol

Catégorie :

Cockpit

Version :

1.0

---

assets/icons/

* 06_Navigation
* 07_FPS
* 08_Systemes
* 09_Urgence
* 10_Vehicules

---

# 🖋️ Typographie

Toutes les icônes utiliseront la même police.

Critères :

* Très lisible
* Compatible avec les petites tailles
* Style technique / industriel
* Majuscules privilégiées

---

# 📏 Règles d'ergonomie

Les commandes critiques seront toujours accessibles rapidement.

Exemples :

* Train d'atterrissage
* VTOL
* Demande d'atterrissage
* Flares
* Noise

Les touches de navigation entre les pages resteront toujours à la même position sur toutes les pages du Stream Deck.

---

# 📚 Documentation

Chaque icône disposera d'une fiche contenant :

* Identifiant
* Nom
* Description
* Fonction Star Citizen
* Raccourci clavier
* Catégorie
* Version
* Historique des modifications

---

# 🚀 Vision du projet

SCCP-FR est conçu pour devenir une bibliothèque graphique et fonctionnelle complète dédiée à Star Citizen.

Le projet devra rester :

* Gratuit
* Ouvert à la communauté
* Documenté
* Facile à maintenir
* Évolutif
